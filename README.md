# nosql
HMIN 313 - NoSQL

## Installation HBase

Télécharger HBASE :
http://apache.crihan.fr/dist/hbase/hbase-0.94.27/

### Configuration

Spécifier le repertoire de Java :
```
export JAVA_HOME="/usr/lib/jre"
```
Ajoutable au fichier conf

## Utilisation

Lancer le serveur :
```
bin/hbase_start
```

Accés au bash Perl :
```
bin/hbase shell
```
